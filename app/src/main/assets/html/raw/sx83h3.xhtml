Aktiwiteit 1: Bevolkings

1. Onderskei tussen die volgende terme: spesie, bevolking en gemeenskap. (6)

  • Spesies is ‘n groep lewende organismes met soortgelyke individue ✓ wat daartoe in staat is om gene te verruil. ✓
  • ‘n Bevolking is deel van ‘n sekere spesie ✓ wat dieselfde habitat op dieselfde tyd bewoon ✓.
  • ‘n Gemeenskap word gevorm deur verskillende spesies✓ wat met mekaar interaksie het in ‘n spesifieke habitat ✓.

2. Verduidelik hoe die digtheid van ‘n bevolking die groeikoers daarvan kan affekteer. (4)

Digtheidsafhanklike faktore oefen ‘n effek uit op ‘n bevolking in verhouding tot sy grootte ✓. Hoe groter die bevolkingsyfer ✓, hoe groter is die invloed wat digtheidsafhanklike faktore sal hê✓. Digtheidsonafhanklike faktore sal ‘n invloed op ‘n bevolking hê ongeag sy grootte. ✓

3. Verskaf die definisie van drakapasiteit / dravermoë. (2)

Dravermoë verwys na grootste getal individue van ‘n biologiese spesie ✓ wat ‘n ekosisteem kan ondersteun oor ‘n onbepaalde tydperk. ✓
